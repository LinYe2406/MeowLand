🐱 猫猫の家服务器规则 v2.4

基本规则 领地系统 商店系统 处罚条例

一、基本原则

  1. 平等原则：欢迎所有玩家，禁止任何形式的歧视行为
  2. 权限等级：操作员 > 管理员 > 普通玩家
  3. 账号限制：每人最多注册3个游戏账号
  4. 严禁使用作弊工具/矿透材质包等第三方程序
  5. 禁止恶意破坏他人建筑或红石装置
  6. 领地保护：必须使用/res指令保护建筑

🏰 领地系统指南

⚠️ 仅限主世界创建领地，其他维度需特别申请

建筑规范

  1. 最小间距：X/Y轴方向48格（3区块）
  2. 禁止在村庄/神庙等系统建筑周边建设
  3. 空中建筑超过10×10需提前审批
  4. 红石设施必须配备开关系统
  5. 公共区域周边建设需提前报备

领地指令手册

指令 说明 示例
/res create [名称] 创建新领地 /res create myhome
/res remove [名称] 删除领地 /res remove oldhome
/res padd [玩家] 添加权限 /res padd myhome Player1

选区流程

  1. 通过菜单获取木锄工具
  2. 左键设置A点，右键设置B点
  3. 在选区内输入/res create [名称]
  4. 通过菜单配置权限

🛒 商店系统指南

实体商店操作

  1. 放置箱子并手持待售物品
  2. 左键点击箱子设置价格
  3. 放入商品完成创建

物品保护机制

右键功能方块（箱子/熔炉等）放置告示牌即可激活保护

快捷商店（Beta）

打开服务器菜单 → 选择"快捷商店"

⚖️ 处罚条例

违规等级 处理措施 申诉时效
轻度 警告+行为记录 7日
中度 封禁7-30天 3日
严重 永久封禁 不可申诉
